# HTML/JS Text Editor

A lightweight, browser-based rich text editor built with HTML, CSS, and vanilla JavaScript.

## Features

- **Rich Text Formatting** - Bold, italic, underline, strikethrough
- **Font Customization** - Multiple font families and sizes
- **Text & Highlight Colors** - Color pickers for text and background
- **Text Alignment** - Left, center, right, and justify
- **Lists** - Ordered and unordered lists
- **Undo/Redo** - Full undo and redo support
- **File Operations** - New, Open, Save, and Print
- **Dark Mode** - Toggle between light and dark themes with saved preference
- **Word & Character Count** - Live status bar with counts
- **Print-Friendly** - Prints only the editor content, hiding all UI chrome
